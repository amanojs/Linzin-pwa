# Next.js & PWA(next-offline) & TypeScript & Prettier & ESLint

## 説明
PWA(Progressive Web Application)対応

### 使用技術等

- React.js
- Next.js
- PWA(next-offline)
- TypeScript
- Prettier
- ESLint

### ライブラリインストール

```
$ yarn install
```

### ローカル実行

```
$ yarn dev
```

上記実行後、以下にアクセス

```
http://localhost:3000
```

### Now へのデプロイ

[ZEIT](https://zeit.co/onboarding)のアカウント未登録の人は作成

Now のインストール

```
$ npm i -g now
```

デプロイ実行

```
$ now
```

上記実行後、生成された URL へアクセス
